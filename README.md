# Modern Basic Calculator

A modern and responsive basic calculator built using HTML, CSS, and JavaScript.  
This project performs standard arithmetic operations with a clean glassmorphism user interface and keyboard support.

---

## 🚀 Features

- Addition, Subtraction, Multiplication, Division
- Percentage Calculation
- Responsive Design
- Modern Glassmorphism UI
- Keyboard Support
- Error Handling
- Clear and Delete Functions
- Smooth Hover Effects

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
